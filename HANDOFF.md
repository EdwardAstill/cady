## End Goal

Build pyseas-cad into a Python package where callers build one domain-neutral
model and export:

- `*.dxf` for 2D drawings,
- `*.stl` for mesh preview/printing,
- `*.step` for basic CAD interchange.

Target public API shape:

```python
from cad import Model, rectangle, circle

plate = rectangle((0, 0), (1.0, 0.6)).with_hole(circle((0.5, 0.3), 0.12))

model = Model("padeye_plate")
model.drawing("front").layer("PLATE").add(plate)
model.part("plate").add(plate.extrude("+z", 0.04))

model.write_dxf("padeye_plate.dxf")
model.write_stl("padeye_plate.stl")
model.write_step("padeye_plate.step")
```

## Notes

- Runtime must remain pure stdlib.
- pyseas-cad must stay domain-blind. No `Padeye`, `Shackle`, or lifting-gear
  objects in core.
- Direct `DxfDrawing` and `StlMesh` APIs should continue to work after
  `Model` is introduced.
- Each stage must end by updating README, roadmap/specs/plans, and preference
  locks if decisions changed.
