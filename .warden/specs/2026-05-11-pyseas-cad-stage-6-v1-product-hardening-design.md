# pyseas-cad Stage 6 - v1 Product Hardening

**Status:** draft.
**Date:** 2026-05-11.
**Purpose:** Turn the working exporters into an adoptable v1 package.

## Goal

Make pyseas-cad easy to install, inspect, verify, and integrate after STEP MVP
lands.

## Scope

- Final README/API documentation pass.
- Gallery index and regeneration instructions.
- Compatibility matrix for DXF/STL/STEP viewers/checkers.
- Packaging metadata and versioning review.
- pyseas-yard integration notes.
- Stale roadmap/ideas cleanup.
- Post-v1 backlog split into explicit follow-up specs where useful.

## Non-Goals

- No new geometry kernel.
- No new export format.
- No major public API redesign unless Stage 5 reveals a blocker.

## Acceptance Draft

- A fresh checkout can install editable and run all examples using documented
  commands.
- Every gallery artifact is generated by a script under `examples/scripts`.
- README explains current support and limitations without relying on internal
  planning files.
- Full test/type/lint suite passes.
- Runtime package metadata still has no dependencies.
