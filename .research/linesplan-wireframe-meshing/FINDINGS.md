# Research Findings: Linesplan Wireframe Meshing

**Date:** 2026-06-25
**Overall confidence:** high for the architecture change, medium for the final
backend choice.

## Key Finding

The correct problem is curve-network surfacing, not DXF-to-mesh conversion and
not generic wireframe triangulation.

The DXF reader should stop returning a mesh generated by hidden heuristics. It
should parse DXF geometry into wire/curve data. A separate surfacing operation
should classify linesplan curves, build a compatible profile/guide network, and
mesh a surface constrained by that network.

## Recommended Algorithm Family

Use Gordon surface / curve-network interpolation as the target algorithm family.

Reason:

- Linesplans are structured crossing curve networks: stations/sections,
  buttocks, waterlines, knuckles, and profile edges.
- Gordon-style methods are designed for ordered profile and guide curves that
  intersect each other.
- CAD-oriented sources describe the necessary pipeline: find intersections,
  sort curves, reparametrize to compatibility, and compute the interpolating
  surface.

## Rejected As Primary Algorithms

- **Current loft between selected sections:** uses only a subset of source
  wires, so guide curves can appear outside the generated faces.
- **Generic ruled surface:** good fallback for ordered non-intersecting section
  curves, but it ignores crossing guide curves.
- **Point-cloud reconstruction:** does not preserve source curves as mesh
  constraints without significant extra machinery.
- **Gmsh/OpenCASCADE patch filling alone:** useful after patch decomposition,
  but not enough to classify and solve a full linesplan network.

## Practical Implementation Direction

Start with a native cady pipeline that produces a triangulated mesh directly
from a compatible sampled curve network. Use `occ_gordon` or `geomdl` only as
optional/reference backends unless dependency approval is given.

This keeps the core repo aligned with the existing runtime rule: no new default
runtime dependencies beyond the current allowed set.
